## Honest caveats (Evidence Contract → Known Blind Spots)

- **单模型**(opencode 默认 build model)。Trigger 准确性在其他模型上可能不同;本 run 验证的是*该模型*对 description 的读法,不是模型无关属性。第二个模型 run 会增强结论。(注:run-001 用 deepseek-v4-pro,run-002 用 opencode 默认 model——两次跑都用单一模型,但非同模型,这是一致的局限。)
- 20 query 是小集合。near-miss 桶(真正的判别测试)共 7 个——全过有意义但不穷尽。
- description-only protocol 是*description 字符串的 unit test*。host 原生 trigger 运行时(opencode/pi 实际如何决定加载一个已装 skill)是独立的集成问题,此处未测,如 eval-guide 所述。
